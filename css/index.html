introducao a css

Seletores

seletores sao padroes usado para escolher elementos html aos quais voce deseja aplicar estilo.

tipos de seletores:class,id,seletor por nome.

propriedade

as propriedade sao carecteristicas especificas que vc deseja alterar, como cor, tamanho da fonte, margem, etc.

Valores

Cada propriedade tem valores que você pode definir para personalizar a aparência. Por exemplo, a cor pode ser definida como "vermelho" ou "#FF0000" para vermelho. O tamanho da fonte pode ser definido em pixels (por exemplo, "16px").

regras CSS

As regras CSS combinam seletores, propriedades e valores para aplicar estilos aos elementos da página. Por exemplo, você pode criar uma regra que diz: "Todos os parágrafos

terão uma cor de texto azul e uma margem de 10 pixels."
